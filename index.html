Garis Garis Pada sisi tertentu

oke disini garis-garis dibagi menjadi 4 bagian tertentu yaitu :

  • border-top ( Atas )
  • border-bottom ( Bawah )
  • border-left ( Kiri )
  • border-right ( Kanan )

Mari kita Bahas satu per satu Pada setiap element Yang ada.

Contoh :
  • Border-top ( Atas )

    • Oke yang yang pertama yaitu border top, Border ini adalah border atas, atau seperti itulah mari kita contohkan pada tulisan ini atau paragraf ini

  • Border-bottom ( Bawah )

    • Okey, yang kedua yaitu border bottom atau border Bawah border ini digunakan pada bagian bawah text seperti ini.

  • Border-Left ( Kiri )

    • Kemudian yang ketiga yaitu Border-left atau border kiri border ini digunakan text pada bagian kiri mari kita contohkan seperti ini

  • Border-right ( Kanan )

    • Okey yang keempat yaitu border kanan atau border right border ini dapat digunakan pada saat text ditulis tapi dalam format kanan,intinya gitu Seperti ini

  • Border-radius ( bulat )

    • Okey yang terakhir yaitu border radius,dimana border ini mengacu bulat pada text ini digunakan pada text tertentu contoh seperti ini
